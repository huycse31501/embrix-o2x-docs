Platform Information

Version: 3.1.9-SNAPSHOT Last Updated: February 2026 Technology Stack: Spring Boot, React, PostgreSQL, RabbitMQ, Redis Architecture: Microservices, Multi-tenant, Event-driven
Key Platform Features:
  • Multi-tenant SaaS platform - Deployment-level multi-tenancy with dedicated databases per tenant
  • Comprehensive Order-to-Cash workflow - Order entry through provisioning, usage, billing, invoicing, payment, and revenue recognition
  • Real-time event processing - Message queues, async workflows, and real-time usage rating
  • Advanced pricing and taxation engines - quote-manager, pricing_sync, tax-engine, and tax-gateway
  • Self-service portal and admin interfaces - Selfcare UI, admin dashboards, and frontend architecture
